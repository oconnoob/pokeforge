# Demo Script (v1)

1. Sign up with email/password on `/auth/signup`.
2. Login and open `/library`.
3. Show built-in Pokemon roster and static front sprites.
4. Open `/generate` and submit a descriptive Pokemon prompt.
5. Wait for pipeline completion and show the generated sprite pair + stats.
6. Return to `/library` and verify the generated Pokemon appears.
7. Open `/battle`, pick your generated Pokemon, and start a 1v1 battle.
8. Play a few turns and narrate log events (status effects, type effectiveness, winner).
