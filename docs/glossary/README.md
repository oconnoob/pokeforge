# Domain Glossary

- Battle Session: A single 1v1 match instance with event history and result.
- Behavior Script: JSON instructions describing Pokemon-specific trigger behavior.
- Built-in Pokemon: Initial seeded Pokemon available to all users.
- Generated Pokemon: User-created Pokemon from natural-language prompt pipeline.
- Move: A selectable battle action with power/type/accuracy metadata.
- Sprite Pair: Front and back 64x64 images tied to a Pokemon.
- Validator: Rule engine that checks generated Pokemon for balance bounds.
